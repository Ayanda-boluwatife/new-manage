# Usage

First===========

Change Directory
cd (name of your app or the folder name)

Second==========

Install dependencies (Tailwind)

```
npm install
```

Third===========

Run the Tailwind CLI to compile the **input.css** during development. The output file is **css/main.css**

```
npm run watch
```

Fourth==========

To build once run...

```
npm run build
```
